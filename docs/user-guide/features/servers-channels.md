# Servers & Channels

Servers are the foundation of Melo - organized communities where people gather around shared interests. Learn how to create, manage, and customize servers and channels.

## ðŸ¢ Understanding Servers

### What is a Server?

A server is like a **digital building** that contains multiple rooms (channels) where community members can:

- Chat in organized topic-based channels
- Join voice and video calls
- Share files and media
- Participate in community events
- Build relationships with like-minded people

### Server Types

**Public Servers**
- Open to everyone through invite links
- Discoverable in server directories
- Great for large communities and interest groups

**Private Servers**
- Invite-only access
- Better for close-knit groups
- Enhanced privacy controls

**Organization Servers**
- Built for teams and workplaces
- Advanced permission structures
- Professional features and integrations

## ðŸ—ï¸ Creating a Server

### Step-by-Step Server Creation

1. **Click the "+" button** in your server list
2. **Select "Create a Server"**
3. **Choose a template** or start from scratch:
   - Gaming Community
   - Study Group  
   - Friends & Family
   - Organization/Business
   - Custom (blank slate)

4. **Configure basic settings:**
   - Server name
   - Server icon (optional)
   - Server description
   - Default language

5. **Set initial permissions:**
   - Who can join without invite
   - Default member permissions
   - Moderation settings

- LFG (Looking for Group) channel
- Bot commands channel

**Study Group Template Includes:**
- Subject-specific channels
- Study session voice channels

- Assignment help channel
- Announcement channel

### Server Settings Overview

Access server settings by:
1. Right-clicking the server name
2. Selecting "Server Settings" (requires permissions)
3. Navigate through the settings categories

### Essential Server Configuration

**General Settings**
- Server name and description

**Moderation Settings**
- Content filtering
- Auto-moderation rules

- Member screening

**Security Features**
- Two-factor authentication requirement
- Verification levels
- DM settings for members
- Raid protection

### Server Roles & Permissions

**Understanding Roles**
- Color-coded member categories
- Hierarchical permission structure
- Customizable permissions
- Role-based channel access

**Common Role Structure:**
1. **Owner** - Full control
2. **Admin** - Nearly full control
3. **Moderator** - Moderation powers
4. **Member** - Standard permissions
5. **New Member** - Limited permissions

**Creating Custom Roles**
1. Go to Server Settings > Roles
2. Click "Create Role"
3. Set role name and color
4. Configure permissions
5. Assign to members
