# Joining Your First Server

Servers are the heart of Melo - they're organized communities where people gather to chat, share, and connect around common interests.

## üè† What is a Server?

A server is like a virtual meeting place with:

- **Multiple channels** for different topics
- **Voice and video rooms** for real-time communication
- **Member roles** that determine permissions
- **Customizable settings** set by server administrators

## üîó Joining with an Invite Link

The most common way to join a server is through an invite link.

### Step 1: Get an Invite Link

Invite links look like:
- `https://melo.app/invite/abc123`
- `melo.app/invite/xyz789`
- Or just the code: `abc123`

You might receive these from:
- Friends or colleagues
- Social media posts
- Community websites
- Email invitations

### Step 2: Use the Invite

**Option A: Click the Link**
1. Click the invite link
2. You'll see a preview of the server
3. Click **"Join Server"**
4. Accept any server rules if prompted

**Option B: Manual Entry**
1. Click the **"+"** button in your server list
2. Select **"Join a Server"**
3. Paste the invite link or code
4. Click **"Join"**

## üîç Finding Public Servers

Some servers are publicly discoverable:

### Server Discovery

1. Click the **"+"** button in your server list
2. Select **"Discover Servers"**
3. Browse categories like:
   - **Gaming**
   - **Education**
   - **Technology**
   - **Art & Creativity**
   - **Community**
4. Click **"Join"** on servers that interest you

### Search by Interest

Use the search bar to find servers by:
- Keywords (e.g., "coding", "music", "gaming")
- Game titles
- Professional topics
- Hobbies and interests

## üë• Server Rules & Guidelines

When you join a server:

### Read the Rules

- Most servers have a **#rules** or **#welcome** channel
- Read these carefully to understand expectations
- Some servers require you to react to rules before accessing channels

### Common Server Rules

- Be respectful to all members
- Stay on-topic in designated channels
- No spam or excessive self-promotion
- Use appropriate language
- Respect voice chat etiquette

## üéØ Your First Steps in a New Server

Once you've joined:

### 1. Introduce Yourself
- Look for an **#introductions** channel
- Share a brief hello and your interests
- Be friendly and follow any introduction templates

### 2. Explore Channels
- **Text channels** - for written conversations
- **Voice channels** - for audio/video calls
- **Announcement channels** - for server updates

### 3. Set Your Nickname
- Right-click your name in the member list
- Select **"Change Nickname"**
- Choose how you want to appear in this server

### 4. Customize Notifications
- Right-click the server name
- Choose **"Notification Settings"**
- Set your preferences for mentions and messages

## ‚ö†Ô∏è Safety Tips

### Verify Server Authenticity
- Be cautious of servers requesting personal information
- Official servers often have verification badges
- When in doubt, ask the person who invited you

### Report Issues
- Use the **"Report"** feature for inappropriate content
- Contact server moderators if you have concerns
- You can leave a server at any time if uncomfortable

## üö™ Leaving a Server

If you need to leave a server:

1. Right-click the server icon
2. Select **"Leave Server"**
3. Confirm your decision
4. You'll no longer receive notifications or see the server

> **Note**: You can rejoin later if you have a new invite link.

## ‚ùì Troubleshooting

**Invite link expired?**
- Ask for a new link from the person who invited you
- Links can have expiration dates or usage limits

**Can't access certain channels?**
- You might need specific roles or permissions
- Ask a moderator or check server rules for requirements

**Server seems inactive?**
- Try different times of day when members might be online
- Look for scheduled events or activities
