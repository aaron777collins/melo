### Message Search

**Using the search bar:**
1. Click the search icon or press **Ctrl+F** (Cmd+F on Mac)
2. Type your search terms
3. Use filters like:
   - `from:username` - messages from specific user
   - `in:channel` - messages in specific channel
   - `has:image` - messages with attachments

## ğŸ”” Notification Center

### Understanding Notifications

**Badge indicators:**
- **Red dot** - unread messages
- **Number badge** - unread mention count
- **White dot** - activity in channel

**Mention types:**
- **@username** - direct mention (red highlight)
- **@here** - mentions online members
- **@everyone** - mentions all server members
- **Role mentions** - mentions specific roles

### Managing Notifications

**Per-server settings:**
1. Right-click server icon
2. Select **"Notification Settings"**
3. Choose your preferences:
   - All messages
   - Only @mentions
   - Nothing

**Per-channel settings:**
- Right-click channel name
- Choose notification level
- Set custom sound (if available)
